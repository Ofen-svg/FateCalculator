# Fate Calculator — калькулятор для ролевой (Fate-style)

Десктопное приложение на **C# / WPF (.NET 8)** с базой данных **SQLite**.

## Что внутри
- Вкладка **«Слуги и Мастера»** — список персонажей (база данных), карточка персонажа:
  изображение, Имя, Класс, Рост/Вес, Мировоззрение, параметры (ранги E…EX++),
  расчётные HP / Мана / Урон обычной атаки, навыки и Небесные Фантазмы.
- Вкладка **«Бои»** — можно открыть сколько угодно вкладок-боёв, в каждой —
  добавление любого числа участников, очередь ходов по Ловкости
  (выше ранг — раньше ход), счётчик раунда и текущий ход, использование
  навыков/НФ с тратой маны.
- **Регенерация по реальному времени**: после кнопки «Завершить бой» персонаж
  запоминает момент окончания и копит HP/Ману, пока вы не откроете его карточку
  или не нажмёте «Синхронизировать» — Слуги: 5 HP / 3 Маны в минуту,
  Мастер: 2 HP / 2 Маны в минуту. Текущие HP/Ману можно поправить вручную в
  редакторе персонажа (если бой прошёл без программы).

## Как собрать и запустить (нужна Windows)
1. Установите [.NET 8 SDK](https://dotnet.microsoft.com/download).
2. В папке проекта выполните:
   ```
   dotnet restore
   dotnet build
   dotnet run
   ```
   либо откройте `FateCalculator.csproj` в Visual Studio 2022+ и нажмите «Запуск».
3. При первом запуске рядом с .exe создастся файл `fate_calculator.db` — это база
   персонажей (SQLite), она сохраняется между запусками программы.

## Структура проекта
```
FateCalculator/
├── FateCalculator.csproj
├── App.xaml / App.xaml.cs
├── MainWindow.xaml / .cs            — главное окно (вкладки Персонажи / Бои)
├── Models/
│   ├── Character.cs                 — модель персонажа + регенерация
│   ├── Skill.cs                     — навык / Небесный Фантазм
│   ├── BattleEncounter.cs           — очередь ходов, раунды
│   ├── RankTables.cs                — таблицы HP/Мана/Урон по рангу
│   └── Lists.cs                     — списки классов, мировоззрений, рангов
├── Data/
│   └── Database.cs                  — SQLite (Microsoft.Data.Sqlite)
├── Views/
│   ├── CharacterEditorWindow.xaml/.cs — создание/редактирование персонажа
│   └── BattleControl.xaml/.cs         — UI одной вкладки боя
└── Resources/
    └── FateTheme.xaml               — тёмно-бордовая/золотая тема в духе Fate
```

## Возможные доработки
- Прямой расчёт урона между двумя выбранными целями в бою (сейчас урон/HP
  изменяет ведущий вручную — поле HP можно поправить в карточке персонажа).
- Экспорт/импорт персонажей в JSON-файл.
- Привязка изображений к самому файлу БД (сейчас хранится путь к файлу).
